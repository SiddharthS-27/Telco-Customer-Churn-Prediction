# Telco-Customer-Churn-Prediction
Telco customer churn prediction project using Python, scikit-learn, and MLflow. Built an end-to-end binary classification pipeline with preprocessing, feature engineering, model comparison, cross-validation, and threshold tuning, selecting HistGradientBoosting as the final model.

Refer to telco_churn_project_report for a comprehensive breakdown of the project implementation.
